## CLOTHES
------------------------------------------------------------------------------------------
METHOD		PATH			DESCRIPTION
------------------------------------------------------------------------------------------
GET		/clothes		list clothes in catalog.
GET		/clothes/[id]		show detail of a given clothe.
POST		/clothes		create a new clothe in catalog.
PUT		/clothes/[id]		update a given clothe.
DEL		/clothes/[id]		delete a given clothe.
